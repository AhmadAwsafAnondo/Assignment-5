What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents a webpage's structure as a tree of objects, allowing programming languages like JavaScript to manipulate HTML and CSS. Through the DOM, developers can dynamically modify content, structure, and styling of web pages.

How do you select on element from the DOM?

To select an element from the DOM, use document.querySelector() for a single element by CSS selector or document.getElementById() for a specific ID.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the DOM is a technique where a single event listener is added to a parent element to handle events from its child elements. This is useful for efficiently managing dynamic content, reducing memory usage, and avoiding multiple event listeners by utilizing event bubbling to catch events from children.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes using the DOM, use setAttribute() and getAttribute(). For styles, modify the style property directly (e.g., element.style.color = "red";). You can also use classList to add, remove, or toggle CSS classes (e.g., element.classList.add("active");).
